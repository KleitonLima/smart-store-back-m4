# Descrição

Repositório da API da Smart Store, utilizando o [Nest](https://github.com/nestjs/nest) framework e TypeScript.

## Clonando o repositório

```
git clone https://github.com/KleitonLima/smart-store-back-m4.git
```

## Instalação

```bash
$ npm install
```

## Rodando a aplicação

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
